Title sample creation
We will have you create a title scene sample using the techniques you have learned so far.
Basically, any concept is fine.
Read the following rules and start creating.

Notes on using folders
・Do not edit folders outside the Create Area.

・Create folders in the following hierarchy and work on them.

Assets -> 03_Create Area -> Hashimoto

Name folders are written in Roman characters.
You are free to manage folders within the name folder.

・Always name the script name, class name, and scene name with Hashimoto at the end.
Example:
	GameManager_Hashimoto
	PlayerController_Hashimoto
	Title_Hashimoto

・If there are any assets you would like to use, please ask us.

・Collaboration is possible. Please assign people in charge and create freely.

If there is anything you do not understand, please ask us immediately.
I will also come and check on you.
For now, there is no specified period, so let's work within the scope of what you can do according to the class.
